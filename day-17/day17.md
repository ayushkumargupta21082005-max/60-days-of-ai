# 🚗 Day 17 – AI Fuel Analytics Dashboard

An AI-powered dashboard that analyzes vehicle fuel data from a CSV file and generates a beautiful, interactive HTML report with cost, CO₂ emissions, maintenance, refueling insights, and an E85 fuel comparison.

Built using a single prompt with **Claude AI**, demonstrating how AI can transform raw CSV data into a production-ready analytics dashboard.

---

## ✨ Features

* 📊 Reads vehicle data directly from a CSV file
* 💰 Calculates average fuel cost per kilometer
* 🌍 Computes CO₂ emissions per kilometer
* 🔧 Analyzes maintenance cost per kilometer
* ⛽ Calculates average refuel/recharge time
* 🚘 Groups vehicles into age buckets
* 📈 Displays Cost vs Vehicle Age trends
* ⚡ Performs complete E85 fuel analysis

  * Pump Savings
  * Running Penalty
  * Break-even Price
  * E85 Score (/10)
* 📉 Interactive SVG Charts
* 🌙 Premium Glassmorphism UI
* 📱 Fully Responsive Design
* ⚙️ Pure HTML, CSS & JavaScript (No Frameworks)

## 🚀 How It Works

1. Upload a CSV containing vehicle fuel data.
2. Enter your vehicle details.
3. Claude AI analyzes the dataset.
4. All metrics are calculated automatically.
5. A complete responsive HTML dashboard is generated.
