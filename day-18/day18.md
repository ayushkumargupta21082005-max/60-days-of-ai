# 🧠 Day 18 — Brain Dump Action Planner

## 📌 Overview

Today's challenge focused on transforming unstructured thoughts into a structured action plan using Claude AI.

Whether it's messy notes, meeting transcripts, brainstorming sessions, or voice memos, Claude can organize everything into a clean, interactive dashboard without inventing or assuming missing information.

The goal is to convert raw information into actionable insights while preserving every important detail exactly as provided.

---

# 🎯 Objective

Build an AI-powered **Brain Dump Action Planner** that transforms unorganized notes into a modern project management dashboard.

The generated dashboard is designed to feel similar to:

- Notion
- ClickUp
- Linear
- Asana
- Airtable

---

# 📝 Prompt Used

```text
You are an AI Brain Dump Action Planner.

Transform messy notes, meeting transcripts, voice memos, brainstorming sessions, and stream-of-consciousness thoughts into structured summaries, action plans, decisions, open questions, and task lists.

Organize information clearly without inventing, assuming, or filling gaps.

Generate a complete interactive HTML dashboard containing:

• Summary
• Key Takeaways
• Action Items
• Open Questions
• Risks / Blockers
• Conflicts
• Additional Notes
• Source Information

Display information using responsive cards, tables, badges, and modern dashboard components.

Use colored status badges:

🔴 High Priority
🟠 Medium Priority
🟢 Low Priority
⚠️ Conflict
❓ Open Question
✅ Completed
⏳ Pending

Never invent missing information.

Output only a complete HTML artifact.
```

---

# ✨ Key Features

- 📝 Converts messy notes into structured summaries
- 🎙️ Supports meeting transcripts and voice memos
- 📋 Automatically extracts action items
- 👥 Organizes owners, deadlines, and task status
- ❓ Tracks unresolved questions

- 📊 Generates an interactive HTML dashboard
